main - интерпретатор python-подобного языка
Usage: ./bin/main your_file

mainLA - лексический анализатор с двумя режимами работы (0 и 1)
Usage: ./bin/mainLA your_file output_format

mainSA - синтаксический анализатор
Usage: ./bin/mainSA your_file

В директории data лежат примеры программ, написанных на данном языке
